# 🤝 Contributing Guide

원활한 협업을 위해 아래 가이드를 준수해주세요.

## 📋 Issue Template (이슈 템플릿)
작업 시작 전, 반드시 **이슈를 먼저 생성**해주세요.

| 템플릿 종류 | 설명 |
| :--- | :--- |
| **✨ Feature Request** | 화면 UI/UX 구현, 기능 개발 |
| **🐞 Bug Report** | 화면 깨짐, 동작 오류 제보 (브라우저 정보 필수) |
| **♻️ Refactoring** | 컴포넌트 분리, 훅 최적화, 코드 구조 개선 |
| **🧹 Chore** | 라이브러리 설치, 환경 설정 등 기타 작업 |

## 🛠 작업 흐름 (Workflow)
1. **이슈 생성**: 템플릿을 선택하여 이슈 등록
2. **브랜치 생성**: `feat/#이슈번호-기능명` 형식으로 생성
3. **개발**: 커밋 컨벤션 준수
4. **PR 생성**: `main` 브랜치로 PR 요청 (스크린샷 첨부 권장)

## 💬 질문
궁금한 점은 [Discussions](https://github.com/Project-Guideon/guideon-frontend/discussions)을 이용해주세요.
